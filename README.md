## 🖥️ Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/ujjwal540/portfolio.git
cd portfolio

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env.local
# Add your ANTHROPIC_API_KEY for the AI chat widget

# 4. Start the dev server
npm run dev
```

Open **[http://localhost:3000](http://localhost:3000)** in your browser.

---

## 📦 Deploy to Vercel

```bash
# Build for production
npm run build

# Deploy via Vercel CLI
vercel --prod
```

> 💡 Or just **push to `main`** — Vercel auto-deploys on every commit.
