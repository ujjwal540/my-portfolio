# Ujjwal Kumar Karn — Portfolio

> Electronics & Communication Engineer building intelligent solutions with AI, Machine Learning, and modern software technologies.

🌐 **Live:** [ujjwalkumarkarn.vercel.app](https://ujjwalkumarkarn.vercel.app)

---

## 🚀 Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **AI Chat:** Claude API (Anthropic)
- **Deployment:** Vercel

---

## ✨ Features

- Dark / Light mode with smooth transitions
- Animated grid background + cursor glow effect
- Typing hero with role cycling animation
- Glassmorphism cards throughout
- Floating right-side control panel
- AI-powered chat widget (asks about me)
- Animated skill bars with scroll reveal
- 6 featured project cards with metrics
- Vertical experience timeline
- Premium contact form
- Fully responsive — mobile first
- SEO optimized

## 🛠️ Run Locally

```bash
# Clone the repo
git clone https://github.com/ujjwal540/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## 📦 Deploy

```bash
# Build for production
npm run build

# Deploy to Vercel
vercel --prod
```

Or just push to `main` — Vercel auto-deploys on every commit.
